# Security Policy

## Supported Versions

We support security updates for the following versions of AquaSense:

| Version | Supported |
| ------- | --------- |
| v0.1.x  | Yes       |

## Reporting a Vulnerability

We take the security of AquaSense seriously. If you find a security vulnerability in this project, please do not report it via public GitHub issues. Instead, please report it via the security reporting mechanism of the repository or contact the maintainers directly.

Once a vulnerability is reported, we will:
1. Acknowledge receipt of the report within 48 hours.
2. Work on a resolution or mitigation.
3. Coordinate a release to address the issue before making details public.
